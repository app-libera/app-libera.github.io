LiberaChat WordPress Plugin

The LiberaChat plugin for WordPress allows you to use in your wordpress web page, the services that the Libera.Chat network

The LiberaChat plugin is an online chat client, your IRC client based on kiwiirc

Installation

From Dashboard ( WordPress admin )

1. Plugins -> Add New

2. Search for LiberaChat

3. Click on Install Now and then Active.

4. Place shortcode in your pages or posts:

[liberachat]

You can specify channel for a specific page instead of using the default channel configured with:

[liberachat chan=#YourChannel]

using FTP or similar

Unzip liberachat.zip file and

Upload liberachat folder to the /wp-content/plugins/ directory.

Activate the plugin through the Plugins menu in WordPress.

Place shortcode in your pages or posts:

[liberachat]

You can specify channel for a specific page instead of using the default channel configured with:

[liberachat chan=#YourChannel]
